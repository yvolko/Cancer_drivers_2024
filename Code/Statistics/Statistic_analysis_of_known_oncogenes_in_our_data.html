Statistics of known oncogenes from COSMIC and oncoKB in our data

Data preparation

Read data and leave only regions with known genes (data has been merged with oncogene containing datasets COSMIC and oncoKB + region thresholding and interval merging is done)

data <- read.csv("../Data/Processed_data/Table_expression_COSMIC_oncoKB_merged_intervals.csv")
data <- data[data$gene_name!="", ]

cancer_types <- read.csv("../Data/Raw_data/Table_for_cancer_types.csv")
cancer_types <- cancer_types %>% 
  select("name", "cancer_type")
data <- left_join(data, cancer_types, join_by("sample" == "name"))

Number os samples per cancer type has been precomputed on our data before. Leave only cancer types where at least 20 high copy regions are present:

n_copies_cancer_types <- read.csv("../Data/Raw_data/Number_of_copies_per_sample_for_cancer_type.csv")

summary_samples_cancer <- n_copies_cancer_types %>%
  group_by(Cancer_type) %>%
  summarize(
    samples = sum(Sample_number)
  ) %>%
  ungroup()

threshold_of_N_samples <- 20
cancers_to_keep <- summary_samples_cancer$Cancer_type[summary_samples_cancer$samples >= threshold_of_N_samples]

data <- data[data$cancer_type %in% cancers_to_keep, ]

Note: 17 out of 28 cancer types are left after this step (kept: BLCA BRCA CESC COAD ESCA GBM HNSC LGG LIHC LUAD LUSC OV PAAD SARC SKCM STAD UCEC)

Acessing distribution of region length and copy number for regions with at least one oncogene and without

Of note: there are 3 regions that are exactly having the same borders within the same cancer type, but they are coming from different samples, their copy number is different. I take therefore mean of copy number. For other region it doesnt make any difference as they have only one sample.

It is:

GBM 7 : 55225629 - 55798100 has oncogene 22 and 42 copies

STAD 11 : 34666863 - 35709082 no oncogene 25 and 27 copies

UCEC 17 : 37671490 - 37974720 has oncogene 16 and 21 copies

count_per_region$coordinates_2 <- sub(".* : ", "", count_per_region$coordinates)
count_per_region <- separate(count_per_region, coordinates_2, into = c("start", "end"), sep = " - ")

If there are multiple regions are they likely to have one or many oncogenes

data$coordinates <- sub(".*: ", "", data$ID)

cancer_sample_summary <- data %>%
  group_by(cancer_type, sample, coordinates) %>%
  summarize(
    in_COSMIC = sum((Tier.COSMIC == "1" | Tier.COSMIC == "2") & OncoKB.Annotated.oncoKB != "Yes"),
    in_oncoKB = sum(OncoKB.Annotated.oncoKB == "Yes" & Tier.COSMIC == "No_data"),
    in_both = sum((Tier.COSMIC == "1" | Tier.COSMIC == "2") & OncoKB.Annotated.oncoKB == "Yes"),
    nowhere = sum(Tier.COSMIC == "No_data" & (OncoKB.Annotated.oncoKB == "No_data" | OncoKB.Annotated.oncoKB == "No")),
    regions_count = n(),
  ) %>%
  ungroup()

cancer_sample_summary <- cancer_sample_summary %>%
  mutate(oncogenes_total = in_COSMIC + in_oncoKB + in_both)
